Welcome to PeerTutor!

Currently in development stage. Check back later for an official release!

In order to use the API, use the 'curl' method in terminal

Here are currently allowed API Requests:

Look up a user (GET) @ /api/user/#

Look up a course (GET) @ api/course/#

Find matches by tutor id (GET) @ /api/findmatchbytutee/#

Find matches by tutee id (GET) @ /api/findmatchbytutor/#

Find course requests by id (GET) @ /api/findcourserequestsbyid/#

Register a user (POST) @ curl http://hackleypeertutor.herokuapp.com/register -d "username=example2" -d "email=example2@gmail.com" -d "grade=11" -d "password=examplePassword" -d "frees=d1p2d3p5d5p6" -X POST -v

Request a course for a user (POST) @ curl http://hackleypeertutor.herokuapp.com/request -d "user_id=1" -d "course_id=2" -d "relationship=True" -X POST -v where relationship True = teach and relationship False = learn

Login user (POST) @ curl http://hackleypeertutor.herokuapp.com/login -d "username=example" -d "password=123" -X POST -v
