PeerTutor API Documentation

Currently in development stage. Check back later for an official release!

In order to use the API, use the 'curl' method in terminal

Here are currently allowed API Requests:

Look up a user (GET) @ /api/user/#

Look up a course (GET) @ /api/course/#

Find matches by tutor id (GET) @ /api/findmatchbytutee/#

Find matches by tutee id (GET) @ /api/findmatchbytutor/#

Find course requests by id (GET) @ /api/findcourserequestsbyid/#

Register a user (POST) @ curl http://hackleypeertutor.herokuapp.com/api/register -d "username=example2" -d "email=example2@gmail.com" -d "grade=11" -d "password=examplePassword" -d "frees=d1p2d3p5d5p6" -X POST -v

Request a course for a user (POST) @ curl http://hackleypeertutor.herokuapp.com/api/request -d "user_id=1" -d "course_id=2" -d "relationship=True" -X POST -v where relationship True = teach and relationship False = learn

The following commands only work with a URL Session of some kind

(Because they rely on flask-login)

Login user (POST) @ curl http://hackleypeertutor.herokuapp.com/api/login -d "username=example" -d "password=123" -X POST -v

Search for current logged-in user (GET) @ /api/user

Logout user (POST) @ /api/logout
